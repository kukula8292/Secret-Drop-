# Secret Drop

## Project Overview
Secret Drop is a project designed to facilitate secure file sharing. The platform allows users to send files securely while ensuring data privacy and integrity.

## Features
- **Secure File Sharing**: Users can upload files and share them securely with others.
- **Encryption**: All files are encrypted during upload and download to ensure data confidentiality.
- **User Authentication**: Supports user accounts for secure access.
- **Tracking and Auditing**: Keeps track of shared files and access history for auditing purposes.

## Usage
1. **Installation**: Clone the repository and install the necessary dependencies.
   ```bash
   git clone https://github.com/kukula8292/Secret-Drop-
   cd Secret-Drop-
   npm install
   ```
2. **Running the Project**: Start the server by running:
   ```bash
   npm start
   ```
3. **Accessing the Application**: Open your web browser and go to `http://localhost:3000`.

4. **Sharing Files**: To share a file, log in to your account, navigate to the upload section, and select your file to share.
